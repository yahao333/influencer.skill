# Influencer.skill

Distill an influencer's content style, interaction patterns, and personal brand into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/influencer` | Invoke the Influencer Skill creator |

## Description

Influencer.skill is a Claude Code extension that distills an influencer's content style, interaction patterns, and personal brand into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, content style
2. **Import Materials**: Video/image posts, comment interactions, fan feedback, etc.
3. **Analyze & Generate**: Extract content style, interaction patterns, brand tone
4. **Confirm & Write**: Generate Influencer Skill file

### Material Types

- Video/image content
- Comment interactions
- Fan feedback
- Self-description
- Paste text directly
